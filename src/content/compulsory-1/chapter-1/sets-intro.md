## 集合的概念

一般地，把一些**能够确定的、不同的**对象看成一个整体，就说这个整体是由这些对象的全体构成的**集合**（set）。

### 集合的表示方法

1. **列举法**：把集合的元素一一列举出来，写在花括号内
   - 例如：$A = \{1, 2, 3\}$

2. **描述法**：用集合中元素的公共特征来描述
   - 例如：$B = \{x \mid x > 0\}$

### 常用数集

| 集合符号     | 含义                   |
| ------------ | ---------------------- |
| $\mathbb{N}$ | 非负整数集（自然数集） |
| $\mathbb{Z}$ | 整数集                 |
| $\mathbb{Q}$ | 有理数集               |
| $\mathbb{R}$ | 实数集                 |

### Python 验证

使用 Python 的 `set` 类型来验证集合的基本操作：

```python
# 定义集合
A = {1, 2, 3, 4, 5}
B = {3, 4, 5, 6, 7}

# 集合的基本运算
print("并集:", A | B)
print("交集:", A & B)
print("差集:", A - B)
print("对称差:", A ^ B)
```
