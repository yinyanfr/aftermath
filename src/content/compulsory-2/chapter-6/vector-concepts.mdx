# 向量的概念

## 学习目标

学完这一页后，你应该能：

1. 理解向量为什么既要看大小，又要看方向；
2. 区分数量与向量；
3. 正确说出向量的模、零向量、单位向量；
4. 判断两个向量是否相等、是否平行；
5. 用有向线段表示向量并读出它的基本信息。

## 为什么要学这个内容

初中里我们经常只关心“多长”“多大”，例如路程是 3 千米、面积是 5 平方米。这些量只需要一个数就能说明白。

但现实中还有很多量，只有大小还不够，比如“向东走 3 千米”和“向西走 3 千米”长度一样，效果却完全不同。

这时就需要一种新的数学对象，同时记录**大小**和**方向**，它就是向量。

## 预备知识

这一页默认你已经会：

1. 线段长度的概念；
2. 平行、共线、同向、反向这些几何语言；
3. 用字母表示点和线段。

## 什么是向量

既有**大小**又有**方向**的量，叫作**向量**。

常见的向量背景有：

- 位移；
- 速度；
- 力。

只有大小没有方向的量，叫作**数量**，例如质量、面积、体积、温度。

向量通常用有向线段表示。例如，有向线段 $\overrightarrow{AB}$ 表示从点 $A$ 指向点 $B$ 的向量。

## 向量的模

向量的大小叫作向量的**模**。

向量 $\overrightarrow{AB}$ 的模记作

$$
\left|\overrightarrow{AB}\right|
$$

它就是线段 $AB$ 的长度。

模只表示“多长”，不表示“朝哪边”。

## 零向量与单位向量

模为 0 的向量叫作**零向量**，记作

$$
\vec 0
$$

零向量的起点和终点重合。它的方向通常看作不确定，因此它与任何向量都平行。

模等于 1 的向量叫作**单位向量**。

注意：单位向量不是只有一个。只要长度为 1，方向可以不同，所以单位向量可以有无数个。

## 相等向量

长度相等且方向相同的两个向量，叫作**相等向量**。

这句话里有两个条件：

1. 模相等；
2. 方向相同。

位置不重要。也就是说，向量可以平移，只要大小和方向不变，仍表示同一个向量。

## 平行向量

方向相同或相反的非零向量，叫作**平行向量**，也叫**共线向量**。

零向量与任意向量都看作平行。

所以：

- 相等向量一定是平行向量；
- 平行向量不一定相等，因为它们可能反向，或者长度不同。

## 直观理解

向量最容易误解的地方是“它到底是不是一条固定的箭头”。

不是。

有向线段只是向量的一种**表示方法**。真正的向量，核心是“长度 + 方向”。

所以同一个向量可以画在平面上的很多不同位置。

这和函数图象上的点不一样。点一移动就不是原来的点了，但向量平移后可能还是同一个向量。

## 推导或说明

为什么“位置不重要”是合理的？

因为向量主要描述的是“变化”或“位移”。

例如：

- 从教室向东走 2 米；
- 从操场向东走 2 米。

这两个动作发生地点不同，但它们的“位移效果类型”相同，都是“向东 2 米”。这就是我们把它们看成同一个向量的原因。

## 例题 1

下列各量中，哪些是向量，哪些是数量？

1. 速度 60 千米/时，方向向北；
2. 教室面积 48 平方米；
3. 位移 5 米，方向向东；
4. 温度 20 摄氏度。

### 讲解

看一个量是不是向量，只要问两个问题：

1. 是否有大小；
2. 是否还必须说明方向。

第 1、3 个量必须说明方向，所以是向量。

第 2、4 个量只需要一个数值，不需要方向，所以是数量。

## 例题 2

已知 $|\vec a|=1$，$|\vec b|=0$，$|\vec c|=3$，并且 $\vec a,\vec c$ 方向相同。

判断：

1. 哪个是单位向量；
2. 哪个是零向量；
3. $\vec a$ 与 $\vec c$ 是否平行；
4. $\vec a$ 与 $\vec c$ 是否相等。

### 讲解

由模可知：

- $|\vec a|=1$，所以 $\vec a$ 是单位向量；
- $|\vec b|=0$，所以 $\vec b$ 是零向量。

$\vec a$ 与 $\vec c$ 方向相同，所以它们平行。

但它们的模分别是 1 和 3，不相等，所以不是相等向量。

## 方法总结

判断基础概念时按下面顺序最稳：

1. 先看这个对象是否同时需要大小和方向；
2. 再看模是多少；
3. 再比较方向关系；
4. 最后判断是否相等。

## 常见错误

1. 认为向量就是一条固定位置的有向线段；
2. 认为单位向量只有一个；
3. 把“平行向量”和“相等向量”混为一谈；
4. 忘记零向量与任意向量都平行；
5. 只比较长度，不比较方向。

## 练习 1

下列各量中，哪些是向量：速度、质量、加速度、面积、力、位移？

### 答案

向量是：速度、加速度、力、位移。数量是：质量、面积。

### 讲解

关键标准是“是否必须带方向”。

速度、加速度、力、位移都不能只说大小，还要说方向，所以它们是向量。

质量、面积只有大小，没有方向，是数量。

## 练习 2

已知 $|\vec a|=4$，$|\vec b|=1$，$|\vec c|=0$。判断 $\vec a,\vec b,\vec c$ 中哪些可能是单位向量，哪些一定不是。

### 答案

$\vec b$ 可能是单位向量；$\vec a,\vec c$ 一定不是单位向量。

### 讲解

单位向量的定义只有一个条件：模等于 1。

所以：

- $|\vec b|=1$，符合定义；
- $|\vec a|=4$，长度不对；
- $|\vec c|=0$，它是零向量，不是单位向量。

### Python 实现

<InlinePythonPlayground
  code={`mods = {"a": 4, "b": 1, "c": 0}
for name, value in mods.items():
    print(name, value == 1)
`}
/>

## 练习 3

若 $\vec a$ 与 $\vec b$ 的模相等，能否判断它们相等？为什么？

### 答案

不能。

### 讲解

相等向量需要同时满足：

1. 模相等；
2. 方向相同。

题目只告诉我们模相等，没有告诉方向关系，所以不能判断它们相等。

例如一个向东 2 的向量和一个向西 2 的向量，模相等，但不相等。

### Python 实现

<InlinePythonPlayground
  code={`a = (2, 0)
b = (-2, 0)

mod_a = (a[0]**2 + a[1]**2) ** 0.5
mod_b = (b[0]**2 + b[1]**2) ** 0.5
print(mod_a, mod_b, a == b)
`}
/>

## 练习 4

在平面上画出三个互不相同的位置，使它们都表示同一个向量“向右 3 个单位，向上 1 个单位”。说明为什么这三个向量相等。

### 答案

例如可以取 $\overrightarrow{AB},\overrightarrow{CD},\overrightarrow{EF}$，其中

$$
B-A=(3,1),\quad D-C=(3,1),\quad F-E=(3,1)
$$

则这三个向量相等。

### 讲解

向量是否相等，不看起点在哪里，只看位移效果是否相同。

只要三个有向线段都满足“横坐标增加 3，纵坐标增加 1”，那么它们长度相同、方向相同，所以是相等向量。

这正体现了向量可以平移的思想。

### Python 实现

<InlinePythonPlayground
  code={`pairs = [((0, 0), (3, 1)), ((2, 5), (5, 6)), ((-4, 2), (-1, 3))]

for start, end in pairs:
dx = end[0] - start[0]
dy = end[1] - start[1]
print(start, end, (dx, dy))
`}
/>

## 练习 5

已知非零向量 $\vec a,\vec b,\vec c$ 满足：$\vec a$ 与 $\vec b$ 同向，$\vec b$ 与 $\vec c$ 反向，且 $|\vec a|=|\vec c|$。判断 $\vec a$ 与 $\vec c$ 的关系。

### 答案

$\vec a$ 与 $\vec c$ 反向平行，且模相等，但不相等。

### 讲解

因为 $\vec a$ 与 $\vec b$ 同向，$\vec b$ 与 $\vec c$ 反向，所以 $\vec a$ 与 $\vec c$ 也反向。

又因为 $|\vec a|=|\vec c|$，所以它们长度相等。

这说明它们是大小相等、方向相反的一对向量，也就是互为反向但不是相等向量。

### Python 实现

<InlinePythonPlayground
  code={`a = (3, 0)
c = (-3, 0)

same_length = (a[0]**2 + a[1]**2) == (c[0]**2 + c[1]**2)
opposite_direction = (a[0] == -c[0] and a[1] == -c[1])
print(same_length, opposite_direction, a == c)
`}
/>
