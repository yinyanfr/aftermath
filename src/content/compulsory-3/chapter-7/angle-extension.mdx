# 角的推广

## 学习目标

学完这一页后，你应该能：

1. 理解为什么初中“0° 到 360°”的角不够用；
2. 区分正角、负角、零角；
3. 会判断一个角的终边所在象限；
4. 会写出与某个角终边相同的所有角；
5. 会把旋转、周期和象限判断联系起来。

## 为什么要学这个内容

摩天轮、风扇、电机、时针都在持续旋转。它们转过的角不可能总停在 360° 以内，也不一定只按逆时针转。

如果还把角只理解成“一个不超过周角的几何图形”，很多真实运动就表达不出来。所以高中先把角从“静止的图形”升级成“旋转的过程”。

## 预备知识

你需要先会：

1. 初中对角的基本认识；
2. 平面直角坐标系的四个象限；
3. 360° 是一周角，180° 是平角，90° 是直角。

## 角的推广

把一条射线绕它的端点旋转到另一位置所形成的图形，叫作角。

在这个定义里：

1. 起始位置的射线叫始边；
2. 结束位置的射线叫终边；
3. 逆时针旋转形成的角叫正角；
4. 顺时针旋转形成的角叫负角；
5. 没有旋转形成的角叫零角。

这样定义以后，角的大小就不再限制在 0° 到 360° 之间，而可以是任意实数角度。

## 直观理解

“450°”并不是一个奇怪的新角，它只是“先转 360°，再多转 90°”。

“-60°”也不是“不合法”，它表示从始边出发顺时针转 60°。

所以角的核心不再是图上张开的大小，而是“转了多少”和“朝哪个方向转”。

## 象限角与终边相同角

把角放进平面直角坐标系时，通常约定：

1. 顶点在原点；
2. 始边在 x 轴正半轴上。

这时：

1. 终边落在第一象限，叫第一象限角；
2. 终边落在第二象限，叫第二象限角；
3. 终边落在第三象限，叫第三象限角；
4. 终边落在第四象限，叫第四象限角；
5. 终边在坐标轴上，就不属于任何象限。

如果两个角相差整周，那么它们的终边相同。

用角度制表示时，和角 $\alpha$ 终边相同的所有角都可写成：

$$
\alpha + k\times 360^\circ \quad (k\in \mathbb Z)
$$

## 推导或证明

为什么“终边相同”一定对应加减整周？

因为一周旋转会回到原来的方向。也就是说：

1. 多转 360°，终边不变；
2. 少转 360°，终边也不变；
3. 多转或少转若干个 360°，终边仍然不变。

所以只要两个角的差是 $k\times360^\circ$，它们终边就相同。反过来，若终边相同，说明其中一个角比另一个角多转或少转了若干整周，因此差也一定是 360° 的整数倍。

## 例题 1

判断下列各角的终边所在位置：$45^\circ$，$210^\circ$，$-60^\circ$，$-90^\circ$。

### 讲解

$45^\circ$ 在第一象限。

$210^\circ=180^\circ+30^\circ$，所以在第三象限。

$-60^\circ$ 表示顺时针转 60°，终边在第四象限。

$-90^\circ$ 的终边在 y 轴负半轴上，不属于任何象限。

## 例题 2

写出与 $60^\circ$ 终边相同的所有角，并找出其中落在 $-360^\circ \le \beta < 720^\circ$ 内的角。

### 讲解

所有与 $60^\circ$ 终边相同的角都可写成：

$$
\beta=60^\circ+k\times360^\circ \quad (k\in \mathbb Z)
$$

当 $k=-1,0,1$ 时，分别得到：

$$
-300^\circ,\ 60^\circ,\ 420^\circ
$$

再往下取 $k=-2$ 会得到 $-660^\circ$，超出范围；再往上取 $k=2$ 会得到 $780^\circ$，也超出范围。

## 方法总结

处理任意角题目时，最稳的顺序是：

1. 先看它是正角还是负角；
2. 再把它化到一个整周范围内，比如 0° 到 360°；
3. 再判断终边所在象限或坐标轴；
4. 若要求终边相同角，就写成“原角 + 整周”。

## 常见错误

1. 认为负角“不是真正的角”；
2. 认为第一象限角一定是锐角；
3. 把“终边相同”误认为“角相等”；
4. 忘记坐标轴上的角不属于任何象限。

## 练习 1

判断下列各角的终边所在象限或轴上位置：$135^\circ$，$300^\circ$，$-150^\circ$，$540^\circ$。

### 答案

$135^\circ$ 在第二象限，$300^\circ$ 在第四象限，$-150^\circ$ 在第三象限，$540^\circ$ 在 x 轴负半轴上。

### 讲解

$135^\circ$ 在 $90^\circ$ 和 $180^\circ$ 之间，所以在第二象限。

$300^\circ=360^\circ-60^\circ$，所以在第四象限。

$-150^\circ$ 与 $210^\circ$ 终边相同，所以在第三象限。

$540^\circ=180^\circ+360^\circ$，终边与 $180^\circ$ 相同，在 x 轴负半轴上。

### Python 实现

r = a % 360
if r == 0:

elif r == 180:

pos = "y轴负半轴"
elif 0 < r < 90:
pos = "第一象限"
elif 90 < r < 180:
pos = "第二象限"
elif 180 < r < 270:
pos = "第三象限"
else:
pos = "第四象限"
print(a, r, pos)
`}

## 练习 2

写出与 $-45^\circ$ 终边相同的所有角。

### 答案

$$
\beta=-45^\circ+k\times360^\circ \quad (k\in \mathbb Z)
$$

### 讲解

终边相同角的通式就是“原角加整周”。因为一周是 $360^\circ$，所以直接写成

$$
\beta=-45^\circ+k\times360^\circ
$$

也可以写成

$$
\beta=315^\circ+k\times360^\circ
$$

两种写法完全等价。

### Python 实现

<InlinePythonPlayground
  code={`base = -45
for k in range(-2, 3):
    print(k, base + 360 * k)
`}
/>

## 练习 3

在 $0^\circ \le \alpha < 360^\circ$ 内，找出与 $-765^\circ$ 终边相同的角。

### 答案

$$
315^\circ
$$

### 讲解

把 $-765^\circ$ 不断加上 $360^\circ$：

$$
-765^\circ+360^\circ=-405^\circ
$$

$$
-405^\circ+360^\circ=-45^\circ
$$

$$
-45^\circ+360^\circ=315^\circ
$$

这时已经落进 $0^\circ \le \alpha < 360^\circ$，所以所求角是 $315^\circ$。

### Python 实现

<InlinePythonPlayground
  code={`angle = -765
print(angle % 360)
`}
/>

## 练习 4

写出终边在第二象限内的角的集合。

### 答案

$$
90^\circ+k\times360^\circ < \alpha < 180^\circ+k\times360^\circ \quad (k\in \mathbb Z)
$$

### 讲解

在一个整周内，第二象限角就是大于 $90^\circ$ 且小于 $180^\circ$ 的角。

再考虑整周重复，只要整体加上 $k\times360^\circ$ 即可，因此得到

$$
90^\circ+k\times360^\circ < \alpha < 180^\circ+k\times360^\circ
$$

### Python 实现

<InlinePythonPlayground
  code={`samples = [100, 150, 470, -170]
for a in samples:
    r = a % 360
    print(a, 90 < r < 180)
`}
/>

## 练习 5

已知角 $\alpha$ 是第二象限角，判断 $\alpha+180^\circ$ 在哪一象限，并说明理由。

### 答案

$\alpha+180^\circ$ 是第四象限角。

### 讲解

第二象限角的终边在 $90^\circ$ 与 $180^\circ$ 之间。

如果再加上 $180^\circ$，终边整体转到对面去：

1. $90^\circ$ 会对应到 $270^\circ$；
2. $180^\circ$ 会对应到 $360^\circ$。

所以新的终边在 $270^\circ$ 与 $360^\circ$ 之间，即第四象限。

### Python 实现

<InlinePythonPlayground
  code={`second_quadrant_examples = [100, 120, 150, 179]
for a in second_quadrant_examples:
    b = a + 180
    print(a, b, b % 360)
`}
/>
