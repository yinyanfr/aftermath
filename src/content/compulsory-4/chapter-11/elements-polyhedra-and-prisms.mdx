# 点、线、面，多面体与棱柱

## 学习目标

1. 知道点、线、面是空间几何体的基本元素；
2. 会用符号表示点线面关系；
3. 理解多面体、棱、顶点、截面等概念；
4. 认识棱柱、直棱柱、正棱柱和平行六面体；
5. 会判断长方体中的平行、异面、垂直关系。

## 为什么要学这个内容

空间几何不是一上来就证明定理，而是先把对象看清楚。点、线、面是语言基础；多面体和棱柱是最常见的立体模型。

## 预备知识

1. 直线、平面、集合符号；
2. 长方体和棱柱的直观认识。

## 点、线、面的关系

空间中：

1. 点在线上，记作 $A\in l$；
2. 点不在线上，记作 $A\notin l$；
3. 直线在平面内，记作 $l\subset\alpha$；
4. 直线与平面相交于一点，记作 $l\cap\alpha=A$；
5. 两平面相交于一条直线，记作 $\alpha\cap\beta=l$。

## 多面体

由若干个平面多边形围成的封闭几何体叫多面体。

相关名称：

1. 面；
2. 棱；
3. 顶点；
4. 面对角线；
5. 体对角线；
6. 截面；
7. 表面积。

## 棱柱

有两个互相平行的底面，其余各面都是平行四边形的多面体叫棱柱。

其中：

1. 侧棱垂直于底面的棱柱叫直棱柱；
2. 底面是正多边形的直棱柱叫正棱柱；
3. 底面是平行四边形的棱柱叫平行六面体；
4. 长方体和正方体都是特殊的直平行六面体。

## 例题 1

为什么长方体一定是棱柱？

### 讲解

因为长方体有两组相对面平行且全等，可以把其中一组看成上、下底面，其余四个面是长方形，也就是平行四边形，所以满足棱柱定义。

## 例题 2

长方体中为什么会出现异面直线？

### 讲解

因为空间中两条直线即使不相交，也未必平行。如果它们不在同一平面内，就叫异面直线。长方体不同面的某些边正好提供了这种典型例子。

## 方法总结

看立体时，先问三件事：

1. 哪些面平行；
2. 哪些边平行；
3. 哪些边不共面。

## 常见错误

1. 把“不相交”直接当成“平行”；
2. 忘记异面直线不共面；
3. 把棱柱的侧面误以为一定是矩形。

## 练习 1

说明圆柱为什么不是多面体。

### 答案

因为圆柱有曲面，而多面体的每个面都必须是平面多边形。

### 讲解

多面体的核心条件是“所有面都是平面多边形”，圆柱侧面不满足这一点。

### Python 实现

<InlinePythonPlayground
  code={`faces = {"圆柱": ["平面", "曲面"], "三棱柱": ["平面", "平面", "平面"]}
print({name: all(face == "平面" for face in kinds) for name, kinds in faces.items()})`}
/>

## 练习 2

为什么直棱柱的侧面都是矩形？

### 答案

因为侧棱垂直于底面，而侧面本来是平行四边形，所以其中一个角成直角后就是矩形。

### 讲解

平行四边形只要有一个角是直角，就成为矩形。

### Python 实现

<InlinePythonPlayground
  code={`angles = [90, 90, 90, 90]
print({"四边形": "平行四边形", "是否有直角": 90 in angles, "可判为矩形": 90 in angles})`}
/>

## 练习 3

说明正方体为什么既是正棱柱，又是长方体。

### 答案

因为它的底面是正方形，侧棱垂直于底面，所以是正棱柱；同时六个面都是正方形，所以当然也是长方体。

### 讲解

这是同一个几何体同时满足两类定义的典型例子。

### Python 实现

<InlinePythonPlayground
  code={`cube = {"底面": "正方形", "侧棱垂直底面": True, "六个面": ["正方形"] * 6}
print({"是正棱柱": cube["底面"] == "正方形" and cube["侧棱垂直底面"], "是长方体": len(set(cube["六个面"])) == 1})`}
/>

## 练习 4

为什么平行六面体中，相对的面一定平行？

### 答案

因为它本质上是以平行四边形为底面的棱柱，而棱柱的两个底面天然平行，对应侧面结构也保证了相对面成对平行。

### 讲解

这是棱柱结构在平行四边形底面情形下的直接结果。

### Python 实现

<InlinePythonPlayground
  code={`faces = [("上底", "下底"), ("前面", "后面"), ("左面", "右面")]
print({pair: "平行" for pair in faces})`}
/>

## 练习 5

说明为什么“棱柱的侧面都是平行四边形”是棱柱定义的一部分。

### 答案

因为这正是棱柱区别于一般多面体的关键结构特征。

### 讲解

如果只要求有两个平行面，而不限制其余面的形状，就会包含很多不是棱柱的几何体。

### Python 实现

<InlinePythonPlayground
  code={`solid = {"平行底面": True, "侧面": ["平行四边形"] * 4}
print(solid["平行底面"] and all(face == "平行四边形" for face in solid["侧面"]))`}
/>
