# 空间中的垂直关系

## 学习目标

1. 理解空间中两直线所成角和异面直线夹角；
2. 掌握线面垂直的定义、判定与性质；
3. 理解二面角和平面角；
4. 掌握面面垂直的判定与性质；
5. 会用垂直关系求距离和角度。

## 为什么要学这个内容

平行描述的是“方向相同”，垂直描述的是“方向最不相同”。空间几何中很多最值、距离、体积问题，最后都要回到垂直关系。

## 预备知识

1. 平面几何中的垂直；
2. 平面的基本事实；
3. 平行关系基础。

## 异面直线所成角

空间中两条异面直线，可以通过过一点分别作它们的平行线，转化成同一点出发的两条相交直线。这个角的不大于 $90^\circ$ 的值，就是异面直线所成角。

## 线面垂直

定义：若一条直线与一个平面内任意一条直线都垂直，则这条直线垂直于该平面。

判定思想：若一条直线垂直于平面内两条相交直线，则这条直线垂直于该平面。

性质思想：若两条直线同垂直于一个平面，则它们平行。

## 二面角与面面垂直

两个半平面组成二面角，二面角大小由其平面角度量。

若两个平面所成二面角为 $90^\circ$，则称这两个平面互相垂直。

判定思想：若一个平面经过另一个平面的一条垂线，则这两个平面垂直。

性质思想：若两个平面垂直，那么其中一个平面内垂直于交线的直线垂直于另一个平面。

## 例题 1

为什么“铅锤线垂直地面，墙面沿铅锤线砌起”可以保证墙面和地面垂直？

### 讲解

因为铅锤线是一条垂直于地面的直线，而墙面经过这条直线，所以由面面垂直判定定理可知墙面垂直于地面。

## 例题 2

为什么一条直线在平面内的射影和原直线之间能帮助我们求“线面角”？

### 讲解

线面角定义为原直线和它在平面内射影所成的锐角。这样做的好处是把空间角转成平面角，便于计算。

## 方法总结

空间垂直题常见转化有三类：

1. 线面垂直转成“垂直两条相交线”；
2. 面面垂直转成“平面过一条垂线”；
3. 距离问题转成直角三角形。

## 常见错误

1. 只证一条线垂直就想推出线面垂直；
2. 把线面角和二面角混淆；
3. 忘记取不大于 $90^\circ$ 的角。

## 练习 1

为什么两条都垂直于同一平面的直线一定平行？

### 答案

因为若不平行，就能在同一点作出两条都垂直于同一直线的不同直线，与平面内垂线唯一性矛盾。

### 讲解

这是线面垂直性质定理。

## 练习 2

为什么线面垂直判定里必须要求“平面内两条相交直线”？

### 答案

因为只垂直于两条平行线还不够，无法确定整张平面的方向。

### 讲解

相交两线能确定一个平面，所以只要同时垂直于这两条线，才能推出垂直于整个平面。

### Python 实现

<InlinePythonPlayground
  code={`line_a = (1, 0, 0)
line_b = (0, 1, 0)
line_c = (2, 0, 0)
print({"相交两线张成平面": line_a != line_b, "平行两线不能唯一确定平面": line_a[1:] == line_c[1:]})`}
/>

## 练习 3

说明什么叫二面角的平面角。

### 答案

在二面角棱上一点，分别在两个面内作垂直于棱的射线，这两条射线所成的角就是平面角。

### 讲解

平面角的作用，就是把空间中的面面夹角转化成平面里的普通角。

### Python 实现

## 练习 4

为什么面面垂直判定里常常先找一条“在一个平面内、又垂直于另一个平面”的直线？

### 答案

因为这正是面面垂直判定定理的直接条件。

### 讲解

相较于直接求二面角，找垂线往往更容易操作。

### Python 实现

## 练习 5

说明为什么点到平面的距离一定沿垂线段测量。

### 答案

因为在所有连接该点与平面内点的线段中，垂线段最短。

### 讲解

这和“点到直线距离取垂线段”是完全类似的思想，只是对象从直线换成了平面。

### Python 实现

<InlinePythonPlayground
  code={`import math

point = (0, 0, 3)
candidates = [(0, 0, 0), (1, 0, 0), (2, 2, 0)]
distances = {candidate: round(math.dist(point, candidate), 3) for candidate in candidates}
print(distances)`}
