# 平面的基本事实与平行关系

## 学习目标

1. 记住平面的三个基本事实；
2. 理解两条相交直线确定一个平面；
3. 区分平行直线和异面直线；
4. 掌握线面平行、面面平行的判定与性质思想；
5. 会用“构造平行线”处理简单证明。

## 为什么要学这个内容

空间几何证明不能只凭“看起来像”。要想严格讨论平行关系，首先要知道平面是怎样确定的，然后才能判断线是否共面、面是否相交。

## 预备知识

1. 点线面符号语言；
2. 平行线基本性质；
3. 棱柱和长方体中的直观关系。

## 平面的基本事实

1. 经过不在同一直线上的三点，有且只有一个平面；
2. 如果一条直线上的两个点在一个平面内，那么这条直线在这个平面内；
3. 如果两个不重合的平面有一个公共点，那么它们有且只有一条过该点的公共直线。

由此可推出：

1. 一条直线和这条直线外一点确定一个平面；
2. 两条相交直线确定一个平面；
3. 两条平行直线确定一个平面。

## 平行直线与异面直线

空间中两条不相交直线：

1. 若共面，则平行；
2. 若不共面，则异面。

所以“**不相交**”并不自动等于“**平行**”。

## 线面平行

判定思想：若平面外一条直线平行于该平面内一条直线，则这条直线平行于这个平面。

性质思想：若一条直线平行于一个平面，那么经过该直线的平面和这个平面相交时，交线与原直线平行。

## 面面平行

判定思想：若一个平面内两条相交直线都平行于另一个平面，则这两个平面平行。

性质思想：两个平行平面同时截第三个平面，所得交线平行。

## 例题 1

为什么“过直线外一点可以作且只可以作一个平面与该直线共面”？

### 讲解

取直线上任意两点，再加上直线外这一点，这三个点不共线，因此唯一确定一个平面，而该平面又包含那条直线。

## 例题 2

为什么长方体里会自然出现异面直线？

### 讲解

因为不同面上的某些边既不相交，也不平行，而且无法落在同一个平面中，所以形成异面关系。

## 方法总结

平行证明常见套路：

1. 先证线线平行；
2. 再证线面平行；
3. 再证面面平行；
4. 或反过来利用面面平行推出线线平行。

## 常见错误

1. 把两条不相交直线都当作平行线；
2. 不先证明共面就下结论；
3. 线面平行时误说“与平面内任意直线都平行”。

## 练习 1

为什么通过一条已知直线的平面有无数个？

### 答案

因为在这条直线外可以任选无数个点，每取一个不同的外点，就和原直线唯一确定一个新平面。

### 讲解

直线只提供两个自由度，还不能把平面固定死。

## 练习 2

说明为什么两个平面若有两个公共点，就一定有无数个公共点。

### 答案

因为两点确定一条直线，而这条直线同时在两个平面内，所以整条直线上的点都是公共点。

### 讲解

这正是平面基本事实 2 和 3 联合作用的结果。

### Python 实现

<InlinePythonPlayground
  code={`common_points = [(0, 0), (1, 1)]
slope = (common_points[1][1] - common_points[0][1]) / (common_points[1][0] - common_points[0][0])
print({"两点确定斜率": slope, "同一直线上的更多点": [(2, 2), (3, 3)]})`}
/>

## 练习 3

为什么异面直线不能在任何一个平面内同时出现？

### 答案

因为“异面”的定义就是不共面。

### 讲解

如果它们能同在某个平面内，又不相交，那就应当是平行，而不是异面。

### Python 实现

  code={`line_a = "平面 alpha 内"

## 练习 4

为什么证明线面平行时，常常要先在平面内找一条与这条线平行的直线？

### 答案

因为这正是线面平行判定定理的直接入口。

### 讲解

线面平行很难直接从“没有公共点”入手证明，所以常用“找平行线”来转化。

### Python 实现

<InlinePythonPlayground
  code={`line_direction = (1, 2, 0)

print({"方向成比例": ratio, "可用线线平行判线面平行": True})`}
/>

## 练习 5

说明为什么两个平行平面截同一个平面时，所得交线必平行。

### 答案

如果不平行，就会在第三个平面内相交，从而导致两个原本平行的平面有公共点，矛盾。

### 讲解

这正是面面平行性质定理的核心反证思路。

### Python 实现

<InlinePythonPlayground
  code={`intersection_lines = ["l1", "l2"]
