# 棱锥、棱台与旋转体

## 学习目标

1. 认识棱锥、正棱锥、棱台、正棱台；
2. 理解高、斜高、侧棱等概念；
3. 理解圆柱、圆锥、圆台是怎样由旋转形成的；
4. 知道轴、母线、轴截面等名词；
5. 会区分多面体和旋转体。

## 为什么要学这个内容

棱锥和圆锥类几何体在建筑、包装、容器、机械中都很常见。它们和棱柱最大的不同，是“越往上越收”。

## 预备知识

1. 棱柱结构；
2. 等腰三角形、梯形；
3. 旋转的基本认识。

## 棱锥与棱台

1. 有一个面是多边形，其余各面是有公共顶点的三角形，这样的多面体叫棱锥；
2. 平行于棱锥底面的平面截棱锥，截面与底面之间的部分叫棱台；
3. 正棱锥指底面是正多边形，顶点在底面中心正上方；
4. 正棱台由正棱锥截得，上下底面都是正多边形。

## 旋转体

1. 矩形绕一边旋转得到圆柱；
2. 直角三角形绕一直角边旋转得到圆锥；
3. 直角梯形绕一腰旋转得到圆台；
4. 半圆绕直径旋转得到球面，围成球。

## 轴、母线、轴截面

旋转体里：

1. 固定不动的直线叫轴；
2. 生成侧面的线段位置叫母线；
3. 经过轴的截面叫轴截面。

例如：

1. 圆柱轴截面是矩形；
2. 圆锥轴截面是等腰三角形；
3. 圆台轴截面是等腰梯形。

## 例题 1

为什么圆台可以看成“截圆锥”得到的几何体？

### 讲解

因为用一个平行于圆锥底面的平面去截圆锥，去掉上面的小圆锥后，剩下的部分就是圆台。

## 例题 2

为什么正棱锥的侧面全等？

### 讲解

因为顶点在底面中心正上方，到各底边的几何位置完全对称，所以每个侧面对应的等腰三角形形状一致。

## 方法总结

识别这类几何体时，重点看：

1. 有没有公共顶点；
2. 有没有平行的上下底面；
3. 是平面围成还是由旋转得到；
4. 截面是否具有对称结构。

## 常见错误

1. 把棱台误看成棱柱；
2. 把旋转体侧面当平面；
3. 混淆高和斜高；
4. 看到圆台就忘记它和圆锥的截取关系。

## 练习 1

为什么圆锥不是多面体？

### 答案

因为圆锥侧面是曲面，不是平面多边形。

### 讲解

多面体只允许平面面片，圆锥不满足。

### Python 实现

print({name: "曲面" in "".join(parts) for name, parts in solid_surfaces.items()})`}

## 练习 2

说明正棱台的侧面为什么是全等的等腰梯形。

### 答案

因为它来自正棱锥被平行底面截去一部分后保留下来的对应侧面，结构完全对称。

### 讲解

原来正棱锥的各侧面全等，截去后得到的剩余部分自然也两两全等，而且上下底边平行，所以是等腰梯形。

### Python 实现

<InlinePythonPlayground
  code={`lateral_faces = ["等腰梯形"] * 4
print(all(face == lateral_faces[0] for face in lateral_faces))`}
/>

## 练习 3

为什么圆柱、圆锥、圆台都属于旋转体？

### 答案

因为它们都可以由一个平面图形绕一条直线旋转一周得到。

### 讲解

这是旋转体的定义本身。

### Python 实现

<InlinePythonPlayground
  code={`solids = ["圆柱", "圆锥", "圆台"]
profiles = {"圆柱": "矩形", "圆锥": "直角三角形", "圆台": "直角梯形"}
for s in solids:
    print(s, profiles[s])
`}
/>

## 练习 4

说明圆柱、圆锥、圆台的轴截面分别是什么图形。

### 答案

矩形、等腰三角形、等腰梯形。

### 讲解

只需回到各自生成图形的旋转前原型即可看出。

### Python 实现

<InlinePythonPlayground
  code={`sections = {"圆柱": "矩形", "圆锥": "等腰三角形", "圆台": "等腰梯形"}
print(sections)`}

## 练习 5

为什么球也属于旋转体？

### 答案

因为球面可以由半圆绕其直径旋转一周得到。

### 讲解

这说明球和圆柱、圆锥一样，都可以用“旋转生成”的观点理解。

### Python 实现

<InlinePythonPlayground
  code={`radius = 3
sample_points = [(radius, 0), (0, radius), (-radius, 0)]
print({"二维母线": sample_points, "旋转轴": "直径"})`}
