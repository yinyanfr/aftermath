# 空间几何体与斜二测画法

## 学习目标

1. 区分平面图形和立体图形；
2. 认识直观图的作用；
3. 理解斜二测画法的规则；
4. 能画简单平面图形和长方体的直观图；
5. 知道“横等纵半竖不变”的含义。

## 为什么要学这个内容

空间几何研究的对象本来是立体，但我们在纸上、屏幕上学习时，必须先把立体画出来。直观图是空间思维和纸面表达之间的桥梁。

## 预备知识

1. 长方形、三角形、圆等平面图形；
2. 长方体、圆柱等常见立体；
3. 平面直角坐标系的基本概念。

## 什么是空间几何体

生活中的物体占据空间的一部分。如果只考虑它的形状和大小，就可以把这部分空间抽象成一个**几何体**。

常见几何体有：

1. 长方体；
2. 棱柱；
3. 棱锥；
4. 圆柱；
5. 圆锥；
6. 球。

## 斜二测画法的核心规则

画水平放置图形的直观图时：

1. 取互相垂直的 $x$ 轴、$y$ 轴；
2. 对应画出夹角为 $45^\circ$ 的 $x'$ 轴、$y'$ 轴；
3. 与 $x$ 轴平行的线段长度不变；
4. 与 $y$ 轴平行的线段长度变为原来的一半；
5. 竖直方向长度不变。

这就是常说的：**横等纵半竖不变**。

## 例题 1

为什么正方形的直观图一般不是正方形？

### 讲解

因为原来互相垂直的两个方向，在直观图里被画成了夹角 $45^\circ$ 的方向；而且其中一个方向长度还缩短为一半。所以形状会变成一般平行四边形，而不是正方形。

## 例题 2

为什么圆柱和球的直观图里常看到椭圆或弧线，而不是“真实圆”？

### 讲解

因为直观图的目的是在平面上表现立体感，不是保持所有图形的真实形状。水平圆在直观图里通常画成椭圆，这样才能表现空间方向感。

## 方法总结

画直观图时，先找三个方向：

1. 横向；
2. 纵向；
3. 竖向。

只要这三个方向处理对了，图就不会乱。

## 常见错误

1. 把两个坐标轴仍画成直角；
2. 忘记纵向减半；
3. 竖直边也跟着减半；
4. 遮挡线不处理成虚线或不擦除。

## 练习 1

说明为什么三角形的直观图仍然是三角形。

### 答案

因为三角形只由三个顶点和三条边组成，作斜二测后各顶点仍对应三个点，连线后仍得到三边形。

### 讲解

斜二测会改变边长和夹角，但不会把“三条边围成一个封闭图形”这一结构改变掉。

### Python 实现

<InlinePythonPlayground
  code={`original_edges = 3
projected_edges = 3
print({"原图边数": original_edges, "直观图边数": projected_edges})`}
/>

## 练习 2

说明为什么平行四边形的直观图仍然是平行四边形。

### 答案

因为平行关系在这种画法下保持不变。

### 讲解

斜二测本质上保持了对应方向上的平行性，因此原图中两组对边平行，作图后仍然是两组对边平行。

### Python 实现

<InlinePythonPlayground
  code={`pairs = [((0, 0), (2, 0)), ((1, 1), (3, 1))]
slopes = [0, 0]
print({"原图两边平行": slopes[0] == slopes[1], "直观图仍判平行": True})`}
/>

## 练习 3

解释“横等纵半竖不变”四个词分别对应什么规则。

### 答案

横等：横向长度不变；纵半：纵向长度变一半；竖不变：竖直长度不变。

### 讲解

这句话是斜二测画法最浓缩的口令，记住它就能快速回忆作图规则。

### Python 实现

<InlinePythonPlayground
  code={`rules = {"横": "不变", "纵": "一半", "竖": "不变"}
for k, v in rules.items():
    print(k, v)
`}
/>

## 练习 4

为什么画长方体时，通常要把被遮挡的边改成虚线？

### 答案

为了在平面图上表达前后遮挡关系，增强立体感。

### 讲解

如果所有边都画成实线，读图的人就不容易判断哪条边在前、哪条边在后，空间感会明显变差。

### Python 实现

<InlinePythonPlayground
  code={`edges = {"前棱": "实线", "后棱": "虚线"}
print(edges)`}
/>

## 练习 5

说明为什么正方形的直观图一般不是菱形也不是正方形，而常是普通平行四边形。

### 答案

因为一个方向保长，另一个方向减半，且夹角也变成 $45^\circ$，因此通常既不等边，也不成直角。

### 讲解

菱形要求边长相等，正方形要求边长相等且夹角为直角，而斜二测通常同时破坏了这两点。

### Python 实现

<InlinePythonPlayground
  code={`side_horizontal = 4
side_depth = 2
angle = 45
print({"横向边": side_horizontal, "纵向边": side_depth, "夹角": angle})`}
/>
