# 直线与平面的夹角、二面角

## 学习目标

1. 理解直线与平面所成角；
2. 理解二面角和平面角；
3. 会用方向向量和法向量求线面角；
4. 会用两个法向量求二面角；
5. 能区分“线线角、线面角、面面角”。

## 为什么要学这个内容

空间里最容易混淆的就是“哪个角才是题目要的角”。向量法的优势之一，就是能把这些角的计算统一到夹角和数量积里。

## 预备知识

1. 直线方向向量；
2. 平面法向量；
3. 数量积与夹角。

## 直线与平面的夹角

若直线 $l$ 不平行也不垂直于平面 $\alpha$，则直线与其在平面内射影所成的锐角，叫作直线与平面所成的角。

若 $\vec v$ 是直线方向向量，$\vec n$ 是平面法向量，夹角记为 $\theta$，则

$$
\sin\theta=\frac{|\vec v\cdot\vec n|}{|\vec v||\vec n|}
$$

## 二面角

两个半平面组成二面角。二面角大小用它的平面角度量。

若两个相交平面的法向量分别为 $\vec n_1,\vec n_2$，二面角大小记为 $\varphi$，则常可先求 $\langle\vec n_1,\vec n_2\rangle$，再转化成所需的锐角二面角。

## 例题 1

若 $\vec v=(0,1,-1)$，$\vec n=(1,1,1)$，求直线与平面所成角的正弦值。

### 讲解

$$
\vec v\cdot\vec n=0\times1+1\times1+(-1)\times1=0
$$

所以线面角正弦值为 0，说明直线与平面平行或在平面内。

## 例题 2

为什么求二面角时常常先求两个法向量夹角？

### 讲解

因为面面关系本身不容易直接算，但法向量把“平面朝向”变成了普通向量方向，从而能用数量积统一处理。

## 方法总结

1. 线面角用“方向向量 vs 法向量”；
2. 二面角用“法向量 vs 法向量”；
3. 最后注意题目要求的是锐角还是补角。

## 常见错误

1. 直接把方向向量和法向量夹角当作线面角；
2. 求出法向量夹角后没转成锐角二面角；
3. 忘记线面角是锐角或 0°、90° 特例。

## 练习 1

若 $\vec v=(1,0,0)$，$\vec n=(0,1,0)$，求线面角。

### 答案

$$
0^\circ
$$

### 讲解

因为数量积为 0，所以方向向量与法向量垂直，直线平行于平面，线面角为 0°。

### Python 实现

<InlinePythonPlayground code={`print(1 * 0 + 0 * 1 + 0 * 0)`} />

## 练习 2

若 $\vec v=(1,0,0)$，$\vec n=(1,0,0)$，求线面角。

### 答案

$$
90^\circ
$$

### 讲解

方向向量平行于法向量，说明直线垂直于平面。

### Python 实现

<InlinePythonPlayground
  code={`v = (0, 0, 2)
n = (0, 0, 1)
print({"方向是否平行": True, "线面角": 90})`}
/>

## 练习 3

若两个平面的法向量互相垂直，判断二面角大小。

### 答案

锐二面角为 $90^\circ$。

### 讲解

两个平面垂直时，对应二面角是直二面角。

### Python 实现

<InlinePythonPlayground
  code={`n1 = (1, 0, 0)
n2 = (0, 1, 0)
dot = sum(a * b for a, b in zip(n1, n2))
print({"法向量点积": dot, "是否直二面角": dot == 0})`}
/>

## 练习 4

为什么线面角不用“直线和任意平面内直线的夹角”定义？

### 答案

因为平面内直线可变，所成角不唯一。

### 讲解

只有取这条直线在平面内的射影，得到的角才是唯一且稳定的。

### Python 实现

<InlinePythonPlayground
  code={`angles = [20, 45, 70]
projection_angle = 45
print({"任取平面内直线得到的角": angles, "唯一的线面角": projection_angle})`}
/>

## 练习 5

为什么二面角要用平面角来度量？

### 答案

因为二面角是空间图形，而平面角能把它转化为普通角，便于比较和计算。

### 讲解

这和线面角用射影转成平面角是同一种思想。

### Python 实现

<InlinePythonPlayground
  code={`space_problem = "二面角"
plane_angle = 60
print({"空间问题": space_problem, "转化后的平面角": plane_angle})`}
/>
