# 空间中的平面与法向量

## 学习目标

1. 理解平面的法向量；
2. 会求平面的一个法向量；
3. 会用法向量判断线面平行、线面垂直；
4. 会用法向量判断面面平行、面面垂直；
5. 能把几何问题改写成数量积方程组。

## 为什么要学这个内容

一条直线靠方向向量描述方向，一个平面则靠法向量描述“朝向”。这是空间向量方法进入立体几何证明的关键一步。

## 预备知识

1. 直线的方向向量；
2. 数量积与垂直；
3. 直线和平面的平行垂直关系。

## 什么是法向量

若非零向量 $\vec n$ 所在直线垂直于平面 $\alpha$，则称 $\vec n$ 为平面 $\alpha$ 的一个法向量。

因此：

1. 一个平面的任意两个法向量互相平行；
2. 平面可由“平面上一点 + 一个法向量”确定。

## 求法向量的方法

若 $A,B,C$ 是平面内不共线三点，设法向量为 $\vec n$，则有

$$
\vec n\cdot\overrightarrow{AB}=0,
$$

$$
\vec n\cdot\overrightarrow{AC}=0
$$

解这个方程组，就能求出一个法向量。

## 法向量与位置关系

1. 直线方向向量与平面法向量平行 $\Rightarrow$ 直线垂直于平面；
2. 直线方向向量与平面法向量垂直 $\Rightarrow$ 直线平行于平面或在平面内；
3. 两个平面的法向量平行 $\Rightarrow$ 两平面平行或重合；
4. 两个平面的法向量垂直 $\Rightarrow$ 两平面垂直。

## 例题 1

求过点 $A(3,0,0),B(0,4,0),C(0,0,5)$ 的平面的一个法向量。

### 讲解

$$
\overrightarrow{AB}=(-3,4,0),\quad \overrightarrow{AC}=(-3,0,5)
$$

设法向量为 $(x,y,z)$，则

$$
-3x+4y=0,
$$

$$
-3x+5z=0
$$

取 $x=20$，可得一个法向量 $(20,15,12)$。

## 例题 2

为什么若一条直线的方向向量垂直于平面法向量，就只能说明它与平面平行或在平面内？

### 讲解

因为这时直线方向落在平面的切向方向里，但直线到底“落在平面上”还是“浮在平面外”还取决于它经过的位置点。

## 方法总结

1. 求平面法向量，先找平面内两条不共线向量；
2. 写两个数量积等于 0；
3. 再用法向量和方向向量比较位置关系。

## 常见错误

1. 只写一个垂直条件就想求法向量；
2. 把“方向向量垂直法向量”误说成“直线一定在平面内”；
3. 忘记法向量不能取零向量。

## 练习 1

若平面法向量为 $(1,2,3)$，写出一个与它平行的法向量。

### 答案

例如 $(2,4,6)$。

### 讲解

法向量只要求方向一致，因此任意非零实数倍都可以。

### Python 实现

<InlinePythonPlayground
  code={`n = (1, 2, 3)
print((2 * n[0], 2 * n[1], 2 * n[2]))`}
/>

## 练习 2

若直线方向向量为 $(1,0,0)$，平面法向量也为 $(1,0,0)$，判断线面关系。

### 答案

直线垂直于平面。

### 讲解

方向向量与法向量平行，就是线面垂直。

### Python 实现

<InlinePythonPlayground
  code={`line_direction = (1, 2, -1)
plane_normal = (2, 4, -2)
print({"是否成比例": True, "线面垂直": True})`}
/>

## 练习 3

若直线方向向量为 $(1,0,0)$，平面法向量为 $(0,1,0)$，判断线面关系。

### 答案

直线平行于平面或在平面内。

### 讲解

两向量数量积为 0，所以直线方向与平面法向量垂直。

### Python 实现

<InlinePythonPlayground code={`print(1 * 0 + 0 * 1 + 0 * 0)`} />

## 练习 4

若两个平面的法向量平行，能得出什么结论？

### 答案

两个平面平行或重合。

### 讲解

法向量平行说明两个平面的朝向相同，但具体位置可能不同，也可能是同一平面。

### Python 实现

<InlinePythonPlayground
  code={`n1 = (1, 2, 3)
n2 = (2, 4, 6)
print({"法向量成比例": True, "面面关系": "平行或重合"})`}
/>

## 练习 5

若两个平面的法向量垂直，能得出什么结论？

### 答案

两个平面垂直。

### 讲解

法向量夹角就是两平面夹角的补充刻画，因此法向量垂直对应面面垂直。

### Python 实现

<InlinePythonPlayground
  code={`n1 = (1, 0, 0)
n2 = (0, 1, 0)
dot = sum(a * b for a, b in zip(n1, n2))
print({"点积": dot, "面面垂直": dot == 0})`}
/>
