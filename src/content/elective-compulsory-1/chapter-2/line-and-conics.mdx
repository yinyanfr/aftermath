# 直线与圆锥曲线的位置关系

## 学习目标

1. 会通过联立方程判断公共点；
2. 理解“相切”对应什么代数条件；
3. 会处理弦长问题；
4. 会用设而不求方法；
5. 会综合使用韦达定理。

## 为什么要学这个内容

前面学的是“单个曲线长什么样”，这一页学的是“直线怎样和曲线相遇”。这一步会把代数方程和几何位置关系真正连起来。

## 预备知识

1. 直线方程；
2. 椭圆、双曲线、抛物线标准方程；
3. 一元二次方程与判别式。

## 基本思想

要研究直线与圆锥曲线的位置关系，只要联立它们的方程。

1. 有两个不同实数解：有两个公共点；
2. 有一个重根：相切；
3. 无实数解：无公共点。

## 弦长问题

若联立后得到关于 $x$ 或 $y$ 的一元二次方程，其两个根对应交点坐标，可以用：

1. 韦达定理处理两交点和、积；
2. 距离公式求弦长；
3. 设而不求避免解出两个具体点。

## 例题 1

判断直线 $y=2x-2$ 与椭圆 $\frac{x^2}{5}+\frac{y^2}{4}=1$ 是否有公共点。

### 讲解

联立后有两个不同实数解，因此有两个公共点。继续可求得弦长。

## 例题 2

为什么“直线与抛物线只有一个公共点”不一定就意味着相切？

### 讲解

因为有些直线可能穿过抛物线的顶点，代数上只出现一个公共点，但这个点未必对应切线意义。真正的相切要看联立后是否出现重根。

## 方法总结

1. 位置关系题先联立再判别；
2. 弦长题优先设而不求；
3. 相切题常设参数后令判别式为 0。

## 常见错误

1. 看到只有一个公共点就直接说相切；
2. 解出根后忘记回代另一变量；
3. 弦长题强行全求两点坐标，导致运算过重。

## 练习 1

判断直线 $y=-2x+4$ 与抛物线 $y^2=4x$ 是否有公共点。

### 答案

有两个公共点。

### 讲解

联立得一元二次方程，有两个不同实根，因此有两个交点。

## 练习 2

说明直线与双曲线有一个公共点时，为什么不一定相切。

### 答案

因为这个公共点可能是双曲线顶点等特殊点，联立后不一定出现重根意义下的切触。

### 讲解

“只有一个公共点”是几何现象，“相切”还要满足局部方向重合，对应代数上的重根条件。

### Python 实现

<InlinePythonPlayground
  code={`x = sp.symbols("x")
equation = x ** 2 - 1
roots = sp.solve(sp.Eq(equation, 0), x)
print({"公共点个数": 1 if len(set(roots)) == 1 else len(roots), "判切关键": sp.discriminant(equation, x) == 0})`}
/>

## 练习 3

为什么直线与圆、直线与椭圆只有一个公共点时通常可判为相切？

### 答案

因为这两类曲线的局部结构比较“封闭和平滑”，只有一个公共点时对应切触。

### 讲解

这是它们与双曲线、抛物线的差别之一。

### Python 实现

<InlinePythonPlayground
  code={`x = sp.symbols("x")
equation = sp.expand((x - 1) ** 2)
print({"交点重合": sp.solve(sp.Eq(equation, 0), x), "解释": "重根对应相切"})`}
/>

## 练习 4

相切题为什么常用判别式 $\Delta=0$？

### 答案

因为联立后得到的一元二次方程有重根，正对应切点重合成一个交点。

### 讲解

“切”在代数上就是“二次方程两个根合并成一个”。

### Python 实现

<InlinePythonPlayground
  code={`x = sp.symbols("x")
discriminant = sp.discriminant(x ** 2 - 2 * x + 1, x)
print(discriminant)`}
/>

## 练习 5

弦长问题里为什么常用韦达定理？

### 答案

因为很多时候只需要两交点坐标的和、积，而不需要分别求出两个点。

### 讲解

这就是“设而不求”的典型使用场景。

### Python 实现

<InlinePythonPlayground

roots = sp.solve(sp.Eq(x ** 2 - 5 * x + 6, 0), x)
print({"两交点横坐标": roots, "和": sum(roots), "积": roots[0] * roots[1]})`}
