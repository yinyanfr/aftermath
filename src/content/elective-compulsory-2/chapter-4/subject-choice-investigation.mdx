# 了解高考选考科目的确定是否与性别有关

## 学习目标

1. 理解一个完整统计调查的基本流程；
2. 会设计调查对象和调查问题；
3. 会把调查结果整理成列联表；
4. 会用独立性检验做出统计判断；
5. 理解统计结论的边界。

## 为什么要做这个探究

前面学过的统计模型，如果只停留在题目里，会显得很抽象。这个活动把“调查设计、数据整理、统计检验、结论表述”串成了一整条链。

## 预备知识

1. 抽样调查；
2. 频率估计概率；
3. 2×2 列联表与独立性检验。

## 一个完整活动通常包括什么

1. 明确研究问题；
2. 明确调查对象范围；
3. 选择抽样方式；
4. 收集并清洗数据；
5. 建立列联表；
6. 做独立性检验；
7. 写出结论和局限。

## 研究问题怎样提得更好

“选考科目是否与性别有关”这句话还可以进一步具体化，例如：

1. 是研究某一门科目，还是研究某一种选科组合；
2. 是研究全年级，还是研究某一学校；
3. 是否还要控制其他因素，比如未来专业方向。

问题越明确，调查越容易做得扎实。

## 数据整理思路

若研究“是否选择物理”和“性别”是否有关，可把数据整理为：

|      | 选择物理 | 不选择物理 |
| ---- | -------- | ---------- |
| 男生 | $a$      | $b$        |
| 女生 | $c$      | $d$        |

然后做独立性检验。

## 例题 1

某年级抽样调查得到数据：男生中 36 人选择物理、24 人未选择；女生中 18 人选择物理、42 人未选择。判断在显著性水平 0.05 下，是否有把握认为“是否选择物理”与性别有关。

### 讲解

这里

$$
a=36,b=24,c=18,d=42,n=120
$$

于是

$$
\chi^2=\frac{120(36\times 42-24\times 18)^2}{60\times 60\times 54\times 66}\approx 10.91
$$

因为

$$
10.91>3.841
$$

所以有 95% 的把握认为二者有关。

## 结论应该怎样写

正确写法是：

1. 有多少把握认为有关；
2. 数据来自什么样本；
3. 结论只在该调查范围内有意义；
4. 不应把统计相关直接说成因果关系。

## 方法总结

1. 调查前先定义清楚变量；
2. 样本要尽量有代表性；
3. 统计结论要配合背景解释；
4. 把“有关”“无关”“因果”这三个词分清。

## 常见错误

1. 只调查一个班级却想代表所有学生；
2. 问题设计含糊，导致数据没法整理；
3. 把独立性检验结果说成绝对结论；
4. 忽视样本偏差和样本量问题。

## 练习 1

说明为什么“某班调查结果”通常不能直接代表“全市高中生情况”。

### 答案

因为样本范围太小，代表性不足，容易受班级类型、教师安排、学校层次等因素影响。

### 讲解

统计推断要想推广到更大总体，样本必须尽量覆盖总体差异，而单个班级往往做不到这一点。

## 练习 2

若想研究“是否选择化学”和“性别”是否有关，应该把原始数据整理成什么类型的表？

### 答案

整理成 2×2 列联表。

### 讲解

因为两个变量都只有两类：性别可分男、女；化学选择情况可分选、不选。

### Python 实现

<InlinePythonPlayground

## 练习 3

独立性检验的结果显示“有 99% 的把握认为有关”，这句话能不能改写成“100% 一定有关”？

### 答案

不能。

### 讲解

统计结论本身带有显著性水平，意味着仍允许小概率犯错，因此不能说成绝对确定。

### Python 实现

<InlinePythonPlayground
  code={`confidence = 0.99
print({"把握": confidence, "是否等于绝对确定": confidence == 1})`}
/>

## 练习 4

说明为什么“样本有代表性”比“样本数量大”更基础。

### 答案

因为如果抽样本身有偏差，样本再大也可能系统性地偏离总体真实情况。

### 讲解

样本量解决的是随机误差问题，代表性解决的是方向是否偏了的问题。方向一旦错，大样本也会把错误估得很稳定。

### Python 实现

<InlinePythonPlayground
  code={`biased_samples = [0.8, 0.79, 0.81]
print({"样本均值": sum(biased_samples) / len(biased_samples), "说明": "大样本仍可能稳定偏离"})`}
/>

## 练习 5

在这个探究中，为什么不能把“性别与选科有关”直接解释成“性别决定选科”？

### 答案

因为统计相关不等于因果关系，选科还可能受兴趣、成绩、专业目标、学校安排等因素共同影响。

### 讲解

独立性检验只能告诉我们两个变量在样本中是否表现出关联，不能单独证明谁导致了谁。

### Python 实现

<InlinePythonPlayground
  code={`ice_cream = [10, 20, 30]
temperature = [20, 28, 35]
print(list(zip(temperature, ice_cream)))`}
