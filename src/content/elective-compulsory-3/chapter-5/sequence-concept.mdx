# 数列的概念

## 学习目标

1. 理解什么是按照一定次序排列的一列数；
2. 会区分数列的项、首项、末项；
3. 会写简单数列的通项公式；
4. 理解数列可以看成定义域是正整数集子集的函数；
5. 会判断递增数列、递减数列与常数列。

## 为什么要学这个内容

找规律、做预测、研究长期变化，都离不开“有顺序的一串数”。数列就是把这种规律正式写成数学对象。

## 预备知识

1. 正整数；
2. 代数式；
3. 函数的基本概念。

## 什么是数列

按照一定次序排列的一列数，叫做数列。

例如：

$$
\frac12,\frac14,\frac18,\frac1{16},\cdots
$$

就是一个数列。

## 数列中的几个基本词

1. 数列中的每一个数叫做项；
2. 第一个数叫首项；
3. 有穷数列的最后一个数叫末项；
4. 项数有限的叫有穷数列，项数无限的叫无穷数列。

## 数列的一般形式

数列常写成

$$
a_1,a_2,a_3,\cdots,a_n,\cdots
$$

其中 $a_n$ 表示第 $n$ 项，也叫通项。

若能写出 $a_n$ 与 $n$ 的关系式

$$
a_n=f(n)
$$

就得到了数列的一个通项公式。

## 数列与函数的关系

数列可以看成一个特殊的函数：

1. 自变量是正整数；
2. 函数值按 $1,2,3,\ldots$ 的顺序排出来。

所以数列既有代数表达，也能用平面上的点来表示。

## 数列的增减

1. 从第 2 项起，每一项都大于前一项，叫递增数列；
2. 从第 2 项起，每一项都小于前一项，叫递减数列；
3. 各项都相等，叫常数列。

## 例题 1

写出数列 $1,\frac12,\frac13,\frac14,\cdots$ 的通项公式。

### 讲解

第 $n$ 项就是正整数 $n$ 的倒数，所以

$$
a_n=\frac1n
$$

## 例题 2

写出数列 $-1,1,-1,1,-1,\cdots$ 的一个通项公式。

### 讲解

符号交替变化，可以写成

$$
a_n=(-1)^n
$$

因为当 $n=1$ 时得 $-1$，当 $n=2$ 时得 $1$，正好符合。

## 方法总结

1. 先看每一项和序号之间的关系；
2. 分数数列常从分子、分母分别找规律；
3. 正负交替常尝试 $(-1)^n$ 或 $(-1)^{n+1}$；
4. 写出通项后，要代前几项回去检查。

## 常见错误

1. 只看数值大小，不看排列顺序；
2. 通项公式只对前几项成立，却误以为对所有项都成立；
3. 把数列的项和函数自变量混淆；
4. 判断增减时忘了“从第 2 项起逐项比较”。

## 练习 1

写出数列 $2,4,6,8,\cdots$ 的通项公式。

### 答案

$$
a_n=2n
$$

### 讲解

每一项都是对应序号的 2 倍。

### Python 实现

<InlinePythonPlayground code={`print([2 * n for n in range(1, 6)])`} />

## 练习 2

写出数列 $1,3,5,7,\cdots$ 的通项公式。

### 答案

$$
a_n=2n-1
$$

### 讲解

这是按顺序排列的正奇数。

### Python 实现

<InlinePythonPlayground code={`print([2 * n - 1 for n in range(1, 6)])`} />

## 练习 3

判断数列 $5,5,5,5,\cdots$ 属于哪一类数列。

### 答案

常数列。

### 讲解

所有项都相等，所以它既不递增也不递减，而是常数列。

### Python 实现

<InlinePythonPlayground code={`print(all(x == 5 for x in [5, 5, 5, 5]))`} />

## 练习 4

写出数列 $\frac23,\frac35,\frac47,\frac59,\cdots$ 的通项公式。

### 答案

$$
a_n=\frac{n+1}{2n+1}
$$

### 讲解

分子依次是 $2,3,4,5$，即 $n+1$；分母依次是 $3,5,7,9$，即 $2n+1$。

### Python 实现

<InlinePythonPlayground
  code={`print([(n + 1) / (2 * n + 1) for n in range(1, 5)])`}
/>

## 练习 5

已知 $a_n=\frac{n-1}{n}$，判断这个数列是递增还是递减。

### 答案

递增数列。

### 讲解

因为

$$
a_{n+1}-a_n=\frac{n}{n+1}-\frac{n-1}{n}=\frac{1}{n(n+1)}>0
$$

所以它递增。

### Python 实现

<InlinePythonPlayground
  code={`def a(n):
    return (n - 1) / n
for n in range(1, 6):
    print(n, a(n))`}
/>
