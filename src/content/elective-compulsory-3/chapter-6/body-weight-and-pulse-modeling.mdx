# 描述体重与脉搏率的关系

## 学习目标

1. 理解建模活动的基本结构；
2. 会把文字条件逐步翻译为变量关系；
3. 理解比例关系和幂函数关系在建模中的作用；
4. 知道模型要接受数据检验；
5. 体会“合理假设”在数学建模里的位置。

## 为什么做这个活动

这一节不是单纯再做一道题，而是把前面的函数、导数、比例和数据分析揉成一个完整建模过程。

## 预备知识

1. 比例关系；
2. 幂函数；
3. 一元线性回归的基本思想；
4. 对数变换的直观认识。

## 建模思路概览

题目给出的生物学假设可以串成一条链：

1. 体重 $W$ 与体积成正比；
2. 表面积 $S$ 与体积的 $\frac23$ 次方同量级；
3. 维持体温消耗的能量与表面积相关；
4. 单位时间血流量与能量消耗相关；
5. 每次心跳输送血量与心脏大小相关；
6. 脉搏率等于“单位时间总血量 ÷ 每次心跳输送血量”。

## 一个常见的模型结论

沿着这些比例关系推下去，通常会得到脉搏率 $f$ 与体重 $W$ 满足某种幂函数关系

$$
f=kW^{-\alpha}
$$

其中 $k>0,\alpha>0$，而教材给出的生物学背景通常会导向 $\alpha$ 接近 $\frac14$。

也就是说，体重越大，脉搏率通常越低。

## 为什么要做数据检验

理论模型只是“可能合理”，不是“自动正确”。

所以我们还要：

1. 查找样本数据；
2. 做适当变换；
3. 比较模型预测与数据是否大体一致。

## 例题 1

如果模型写成 $f=kW^{-1/4}$，说明它表示怎样的变化趋势？

### 讲解

指数是负数，说明体重增大时脉搏率下降。而且这是下降得越来越平缓的一种幂函数关系。

## 例题 2

为什么常把模型两边取对数后再做线性分析？

### 讲解

因为

$$
f=kW^{-\alpha}
$$

两边取对数后变成

$$
\ln f=\ln k-\alpha \ln W
$$

这就转化成了直线关系，更方便和数据做比较。

## 方法总结

1. 建模先理清变量之间的因果链条；
2. 每一步都要说明假设依据；
3. 最终模型要能被数据检验；
4. 拟合得再好，也要承认模型只是近似。

## 常见错误

1. 跳步太快，直接写结果，不说明推导链条；
2. 把“正比于”误写成“等于”；
3. 没有区分理论模型和数据拟合模型；
4. 用极少样本就下绝对结论。

## 练习 1

若模型是 $f=kW^{-1/4}$，比较体重为 16 和 81 的两类动物，哪一类脉搏率更高？

### 答案

体重为 16 的那一类脉搏率更高。

### 讲解

因为指数为负，体重越大，脉搏率越低，所以体重较小者脉搏率较高。

### Python 实现

<InlinePythonPlayground
  code={`import math
for W in [16, 81]:
    print(W, W ** (-0.25))`}
/>

## 练习 2

说明为什么“体重越大，脉搏率越低”不等于“体重是唯一决定因素”。

### 答案

因为脉搏率还可能受物种、活动状态、年龄、健康状况等因素影响。

### 讲解

模型抓住的是主要趋势，不是把所有影响都排除掉了。

## 练习 3

为什么把幂函数模型取对数后，常更方便做数据分析？

### 答案

因为能把乘方关系变成线性关系。

### 讲解

这能让我们使用散点图、回归直线等熟悉工具来检验模型是否合理。

### Python 实现

<InlinePythonPlayground
  code={`W, k, alpha = sp.symbols("W k alpha", positive=True)
model = k * W ** (-alpha)
print(sp.expand_log(sp.log(model), force=True))`}

## 练习 4

在建模活动中，什么叫“合理假设”？

### 答案

就是在不完全掌握全部细节时，基于背景知识作出的、能帮助建立模型且不过分脱离实际的假设。

### 讲解

合理假设不是随便猜，而是有背景支撑、能被后续检验的简化。

### Python 实现

<InlinePythonPlayground
  code={`def pulse_model(weight, k=180, alpha=0.25):
    return round(k * (weight ** (-alpha)), 2)

weights = [40, 60, 80]
print({weight: pulse_model(weight) for weight in weights})`}
/>

## 练习 5

为什么说建模题的答案通常不止一种？

### 答案

因为变量选择、简化方式、数据处理和拟合方法都可能不同，只要逻辑自洽、结果能解释并能接受检验，都可能形成不同但合理的模型。

### 讲解

建模不是只追求唯一公式，更重视建模过程的清晰性和解释力。

### Python 实现

<InlinePythonPlayground
  code={`weights = [40, 60, 80]
observed = [72, 65, 60]

def power_model(weight):
return 180 \* (weight \*\* (-0.25))

def linear_model(weight):
return 84 - 0.3 \* weight

for name, model in [("幂函数模型", power_model), ("线性模型", linear_model)]:
errors = [round(model(w) - y, 2) for w, y in zip(weights, observed)]
print(name, errors)`}
