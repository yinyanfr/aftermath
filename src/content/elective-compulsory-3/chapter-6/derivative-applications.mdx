# 利用导数解决实际问题

## 学习目标

1. 会把实际问题转化成函数最值问题；
2. 会选择合适的自变量；
3. 会写出目标函数；
4. 会用导数求最优方案；
5. 会解释最优结果的实际意义。

## 为什么要学这个内容

导数最有力量的地方，不只是研究函数图像，更在于帮我们找“最好”“最省”“最快”“最大”“最小”的方案。

## 预备知识

1. 函数最值；
2. 建模中的变量设定；
3. 几何和代数基础。

## 解题基本流程

1. 读题，明确要求优化什么；
2. 设自变量；
3. 写出目标函数；
4. 确定定义域；
5. 用导数求最值；
6. 回答实际问题。

## 例题 1

边长为 1.2 m 的正方形铁板，四角各截去边长为 $x$ 的小正方形，折成无盖盒子。问 $x$ 为多少时体积最大？

### 讲解

体积函数为

$$
V(x)=x(1.2-2x)^2,\quad 0<x<0.6
$$

求导得

$$
V'(x)=12(x-0.2)(x-0.6)
$$

在定义域内，函数在 $x=0.2$ 处达到最大值，所以应截去边长 0.2 m 的正方形。

## 例题 2

某产品总成本函数为

$$
C(Q)=10Q^2+200Q+1000
$$

求每吨平均成本最低时的月产量。

### 讲解

平均成本函数是

$$
f(Q)=\frac{C(Q)}{Q}=10Q+\frac{1000}{Q}+200
$$

求导得

$$
f'(Q)=10-\frac{1000}{Q^2}
$$

令 $f'(Q)=0$，得 $Q=10$。

所以月产量为 10 吨时，每吨平均成本最低。

## 方法总结

1. 优化题的核心是找目标函数；
2. 自变量选得好，函数就容易写；
3. 定义域一定不能漏；
4. 求出最值后，要回到原题语言作答。

## 常见错误

1. 目标函数写错成中间量；
2. 定义域没有限制；
3. 求出临界点后没判断是不是最优；
4. 最后没有回答实际量。

## 练习 1

长方形周长固定为 20，问长和宽分别为多少时面积最大？

### 答案

长宽都为 5。

### 讲解

设长为 $x$，则宽为 $10-x$，面积为

$$
S=x(10-x)
$$

它在 $x=5$ 时取最大值。

### Python 实现

<InlinePythonPlayground
  code={`x = 5
print(x, 10 - x, x * (10 - x))`}
/>

## 练习 2

半径为 3 的圆中作内接矩形，问何时面积最大？

### 答案

当矩形是正方形时面积最大。

### 讲解

设半边长为 $x,y$，则有 $x^2+y^2=9$，面积是 $4xy$。对称性和最值原理都表明 $x=y$ 时最大。

### Python 实现

<InlinePythonPlayground
  code={`import math
x = y = 3 / math.sqrt(2)
print(4 * x * y)`}
/>

## 练习 3

说明为什么最优化题里“设变量”这一步不能省。

### 答案

因为没有变量，就无法把实际问题写成函数，也就无法求导和比较大小。

### 讲解

导数作用在函数上，函数又依赖变量，所以建模必须先把变化量抽出来。

### Python 实现

<InlinePythonPlayground
  code={`x = sp.symbols("x", positive=True)
volume = x * (sp.Rational(6, 5) - 2 * x) ** 2
print(sp.expand(volume))`}
/>

## 练习 4

在实际问题中，为什么找到临界点后还要检查它是否落在定义域内？

### 答案

因为超出定义域的点虽然满足代数方程，但并不对应实际可行方案。

### 讲解

例如长度不能为负，截角边长不能超过原边长的一半，这些都来自现实条件。

### Python 实现

<InlinePythonPlayground
  code={`x = sp.symbols("x")
f = x * (10 - x)
critical_points = sp.solve(sp.Eq(sp.diff(f, x), 0), x)
feasible_domain = [point for point in critical_points if 0 <= point <= 4]
print({"临界点": critical_points, "落在[0,4]内": feasible_domain})`}
/>

## 练习 5

已知某设计方案的成本函数在定义域内先减后增，这说明什么？

### 答案

说明它在中间某处有最低成本点。

### 讲解

先减后增意味着函数先下降再上升，所以中间转折处就是极小值点，也通常是最优方案所在位置。

### Python 实现

<InlinePythonPlayground
  code={`def cost(x):
    return (x - 3) ** 2 + 2

samples = {x: cost(x) for x in [1, 2, 3, 4, 5]}
print(samples)`}
/>
