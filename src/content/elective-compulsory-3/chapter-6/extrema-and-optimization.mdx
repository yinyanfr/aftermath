# 导数与函数的极值、最值

## 学习目标

1. 理解极大值、极小值与最值的区别；
2. 知道导数为 0 是极值的必要条件；
3. 会用导数符号变化判断极值；
4. 会在闭区间上求函数最值；
5. 会结合函数图像理解结果。

## 为什么要学这个内容

最优化问题的核心往往就是求最值，而导数正是求极值和最值最有效的工具之一。

## 预备知识

1. 函数单调性；
2. 导数符号；
3. 区间与端点。

## 极值的判断

若函数在 $x_0$ 附近可导，且

1. $x_0$ 左侧导数为正，右侧导数为负，则 $x_0$ 是极大值点；
2. $x_0$ 左侧导数为负，右侧导数为正，则 $x_0$ 是极小值点。

## 最值的求法

在闭区间上求最值时，通常比较：

1. 各个极值点处的函数值；
2. 区间端点处的函数值。

最大的就是最大值，最小的就是最小值。

## 例题 1

求函数 $f(x)=\frac13x^3-4x+4$ 的极值。

### 讲解

$$
f'(x)=x^2-4=(x-2)(x+2)
$$

所以临界点是 $x=-2,2$。

导数符号变化表明：

1. $x=-2$ 处取得极大值；
2. $x=2$ 处取得极小值。

函数值分别是

$$
f(-2)=\frac{28}{3},\quad f(2)=-\frac43
$$

## 例题 2

求函数 $f(x)=x^2-2x-1$ 在区间 $[-2,0]$ 上的最值。

### 讲解

$$
f'(x)=2x-2<0\quad (x\in[-2,0])
$$

所以函数在该区间上单调递减。

因此：

1. 最大值在左端点 $x=-2$ 处取得，值为 $7$；
2. 最小值在右端点 $x=0$ 处取得，值为 $-1$。

## 方法总结

1. 极值看导数符号变化；
2. 导数为 0 只是候选点，不是充分条件；
3. 最值题要记得检查端点；
4. 先画出导数符号表会更稳。

## 常见错误

1. 一见导数为 0 就认定极值；
2. 只看极值点，漏看区间端点；
3. 单调区间判断对了，但函数值算错；
4. 极值和最值混为一谈。

## 练习 1

求函数 $y=x^2-7x+6$ 的极值。

### 答案

在 $x=\frac72$ 处取得极小值，极小值为 $-\frac{25}{4}$。

### 讲解

$$
y'=2x-7
$$

只有一个临界点 $x=\frac72$，且导数由负变正，所以这里是极小值点。

### Python 实现

<InlinePythonPlayground
  code={`x = 7 / 2
print(x ** 2 - 7 * x + 6)`}

## 练习 2

求函数 $y=-x^3+3x-5$ 的极值点。

### 答案

极小值点 $x=-1$，极大值点 $x=1$。

### 讲解

$$
y'=-3x^2+3=3(1-x^2)
$$

导数在 $x=-1,1$ 处为 0，并且符号变化表明这两个点分别对应极小和极大。

### Python 实现

<InlinePythonPlayground
  code={`x = sp.symbols("x")
f_prime = sp.diff(-x ** 3 + 3 * x, x)
critical_points = sp.solve(sp.Eq(f_prime, 0), x)
print(critical_points)`}
/>

## 练习 3

求函数 $y=x^4-2x^2+5$ 在区间 $[-2,2]$ 上的最小值。

### 答案

$$
4
$$

### 讲解

$$
y'=4x^3-4x=4x(x^2-1)
$$

临界点是 $-1,0,1$，连同端点 $-2,2$ 一起比较函数值，可得最小值为 4，在 $x=\pm1$ 处取得。

### Python 实现

<InlinePythonPlayground
  code={`def f(x):
    return x ** 4 - 2 * x ** 2 + 5
points = [-2, -1, 0, 1, 2]
vals = [(x, f(x)) for x in points]
print(vals)`}
/>

## 练习 4

说明极大值一定比极小值大吗。

### 答案

不一定。

### 讲解

极大值和极小值都是“局部”概念，不同位置的局部高点和低点之间不一定满足全局大小比较。

### Python 实现

<InlinePythonPlayground
  code={`def f(x):
    if x <= 0:
        return -(x + 1) ** 2 - 100
    return (x - 1) ** 2 + 100

print({"局部极大值": f(-1), "局部极小值": f(1)})`}
/>

## 练习 5

已知函数在区间 $[a,b]$ 上连续，为什么求最值时要比较端点值？

### 答案

因为最大值或最小值可能就在端点处取得。

### 讲解

极值点只反映区间内部的局部变化，而端点不属于左右都完整的内部位置，所以必须额外检查。

### Python 实现

<InlinePythonPlayground
  code={`def f(x):
    return x ** 2 - 2 * x - 1

points = [-2, 0, 1]
values = {x: f(x) for x in points}
print(values)`}
/>
